# AWS IAM Users and Groups Lab

## Project Overview

This project demonstrates the fundamentals of AWS Identity and Access Management (IAM). I created and managed IAM users, IAM groups, and permissions following AWS security best practices.

---

## Objectives

- Create IAM users
- Create IAM groups
- Assign users to groups
- Attach AWS managed policies
- Practice least-privilege access
- Understand IAM permission inheritance

---

## AWS Services Used

- AWS IAM

---

## Skills Demonstrated

- Identity and Access Management (IAM)
- User Management
- Group Management
- Permission Management
- AWS Security Best Practices
- Principle of Least Privilege

## What I Learned

This lab strengthened my understanding of AWS Identity and Access Management by demonstrating how users inherit permissions through IAM groups instead of assigning permissions individually. It also reinforced AWS security best practices and the principle of least privilege.
